This library includes several font sizes that you can utilise to change size of your text easily.

#### Usage
```jsx
import { fontSizes, Text } from 'builtin-react';

<Text theme={{
    fontSize: fontSize.typeL
}}>Large Text</Text>
```

#### Available Font Sizes
| Name      | Font Size   |
| --------  | ----------- |
| typeXXL   | 42px        |
| typeXL    | 35px        |
| typeL     | 28px        |
| typeM     | 21px        |
| typeN     | 16px        |
| typeS     | 14px        |
| typeXS    | 12px        |
